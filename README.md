# Dotfiles

Simple dotfiles setup for me.

## How to install

Firstly, clone the repository:

```
git clone https://github.com/oyvindremme/dotfiles.git
```

Second, make create_links.sh executable:

```
chmod +x create_links.sh
```

Thirdly, run the script:

```
./create_links.sh
```

And you're done! Cheers!
